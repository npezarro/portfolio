# Portfolio — CLAUDE.md

## What This Is
Static HTML/CSS/JS portfolio site deployed at /portfolio/ on the GCP VM. No build step, no framework.

## Design
Variant Q ("Magazine Spread") is the production design. Editorial aesthetic with:
- Fonts: Playfair Display (display), Source Serif 4 (body), IBM Plex Mono (accents)
- Colors: Light (#fefdfb bg, #8b1a1a burgundy accent), Dark (#141210 bg, #c43c3c accent)
- CSS custom properties for theming, dark/light toggle with localStorage

## Structure
- `index.html` — Main portfolio page (sections: hero, masthead, 4 project sections, experience, patents, background, footer)
- `assets/style.css` — Full stylesheet
- `assets/site.js` — Dark mode toggle + IntersectionObserver scroll reveals
- `assets/project.css` — Shared styles for project detail pages (uses OLD blue accent, not yet updated)
- `assets/switcher.js` — Variant switcher (hidden on main page, visible on /a/ through /s/)
- `projects/*.html` — 20 project detail pages (generated by generate-pages.js)
- `generate-pages.js` — Node.js script that generates project detail pages
- `a/` through `s/` — 19 design variant subdirectories (reference only)

## Deployment
Static files via SCP to /var/www/html/portfolio/ on the GCP VM. Apache serves directly.
See privateContext for SSH credentials and deploy commands.

## Security
Pre-commit hook blocks the domain name and VM IP in committed code (public repo). Don't put identifying URLs in committed files.
